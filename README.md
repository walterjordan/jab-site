# JAB Site

This is a Next.js application for connecting to and managing Facebook Messenger integrations. It provides tools for businesses to connect their Facebook pages and manage messaging sessions.

## Getting Started

To get started with development, follow these steps:

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Tech Stack

This project is built with the following technologies:

-   [Next.js](https://nextjs.org/) - React framework for building server-rendered applications.
-   [React](https://react.dev/) - JavaScript library for building user interfaces.
-   [TypeScript](https://www.typescriptlang.org/) - Typed superset of JavaScript.
-   [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework.
-   [ESLint](https://eslint.org/) - For code linting and quality.
